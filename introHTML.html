HTML Formatting

In HTML, we have many special elements that can provide special meaning to text content.

<b> Bold the text
<strong> An alternative for <b>, which tell that this text is essential.
<i> Italic the text
<em> Similar to italic but used when we want to emphasize the text.
<mark> It marks the text with a default "yellow" background colour
<small> Decrease the text size
<del> It prints a cross line over the text.
<ins> Represents the inserted text by putting an underline.
<sub> This element is used to display the subscript.
<sup> It can make a text superscript.

HTML colors

HTML colors are specified with predefined color names, or with RGB, HEX, HSL, RGBA, or HSLA values.

Types of Colors

  • Background Color
    • Example <h1 style="background-color:DodgerBlue;">welcome to infobeans Foundation/h1> <p style="background-color:Tomato;"welcome to infobeans Foundation/p>
  • Text Color
    • Example <h1 style="color:Tomato;">welcome to infobeans Foundation /h1> <p style="color:DodgerBlue;">welcome to infobeans Foundation/p>
  • Border Color
    • Example <h1 style="border:2px solid Tomato;"welcome to infobeans Foundation/h1> <h1 style="border:2px solid DodgerBlue;">welcome to infobeans Foundation/h1>
